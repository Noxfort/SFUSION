# 🛠️ Contributing to SFusion Mapper

First off, thank you for considering contributing to SFusion Mapper! It's people like you that make this ecosystem such a great tool.

## Getting Started

1. Familiarize yourself with our [[ARCHITECTURE]].
2. Ensure you have followed the setup guide in [[README]].
3. Review our [[CODE_OF_CONDUCT]] to understand our community standards.

## How to Contribute

### Reporting Bugs
- Ensure the bug was not already reported.
- Open a new issue with a clear title and description.
- Include logs, specifically from `[[src/utils/slm_telemetry.py]]` if applicable.

### Submitting Pull Requests
- Create a new branch for your feature or bugfix.
- Follow the clean code principles outlined in our MVC architecture.
- Update the [[CHANGELOG]] with a summary of your changes.
- Ensure your PR description clearly describes the problem and solution.
